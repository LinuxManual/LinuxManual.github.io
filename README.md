# LinuxManual.github.io

LinuxManual Pro is a professional Linux reference site focused on practical operations work and production-ready documentation.

## Highlights

- 400+ practical Linux commands with instant in-page search and one-click copy.
- Distro intelligence cards with official download links.
- Quick-start operational guides for setup, hardening, troubleshooting, and launch readiness.
- SEO-ready metadata, Open Graph tags, JSON-LD schema, `robots.txt`, and `sitemap.xml`.
- Fast, dependency-free static delivery for GitHub Pages.

## Deployment checklist

- Verify all outbound distro links are still valid and official.
- Re-generate and submit `sitemap.xml` after structural updates.
- Confirm metadata previews (Open Graph/Twitter) before publishing.
- Run local validation (`python -m http.server`) and smoke-test key sections.

## Files

- `index.html` – Main website page.
- `robots.txt` – Crawling rules + sitemap reference.
- `sitemap.xml` – Search engine sitemap.
